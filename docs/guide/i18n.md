# Internationalization (i18n)

Agent Monitor supports Chinese and English with one-click language switching.

## Switching Languages

Click the language toggle button in the top-right corner of the nav bar:
- **EN** -> Click to switch to Chinese
- **中文** -> Click to switch to English

Your language preference is saved to `localStorage` and persists across sessions.

## Supported Languages

| Language | Code | Coverage |
|----------|------|----------|
| English | `en` | Full |
| Chinese (Simplified) | `zh` | Full |

## What Gets Translated

- Navigation labels
- Button text and form labels
- Placeholder text
- Slash command descriptions
- Status messages and system output
- Error messages

## Technical Details

- Translations are in `client/src/i18n/translations.ts`
- Uses React Context (`LanguageProvider`) with a custom `useTranslation()` hook
- No external i18n library needed
- Technical terms (CLI flags, model names, provider names) remain untranslated
